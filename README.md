#프로그램 설명:
틱택토 게임 구동 과정
1. 3x3 크기의 빈 보드를 생성.
2. 플레이어(X)와 컴퓨터(O)가 번갈아가며 보드에 표시.
   - 플레이어는 직접 입력 (행, 열 입력)
   - 컴퓨터는 무작위로 빈 칸을 선택
3. 매 턴마다 승리 조건을 검사.
   - 가로, 세로, 대각선 중 한 줄을 같은 기호로 채우면 승리
4. 누군가 승리하면 게임을 종료하고 결과 출력.
5. 9턴까지 승부가 나지 않으면 무승부로 게임 종료
